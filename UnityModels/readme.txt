Unity 5.6.1f1 was used to create this models.

To run it on computer simply donwload the UnityLabModel file, open it using Unity 5.6.1f1 and click on play button.

To use it with HTC Vive, donwload UnityLabModel_HTC_Vive model and click on play button, it should play on the HTC device.
To teleport around the room use controller's hair trigger.
